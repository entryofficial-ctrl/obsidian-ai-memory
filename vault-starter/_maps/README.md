# Maps

프로젝트 경계를 넘어 서로 관련된 기록을 묶는 Obsidian 지도입니다.  
원본 문서는 그대로 두고, 여기서는 업무 흐름별로 링크를 엮습니다. 작성 규칙은 [[MAP_AUTHORING|지도 작성 가이드]] 참고.

## 지도 목록

| 지도 | 묶는 범위 |
|---|---|
| [[_maps/ai-memory-system|AI-Memory 시스템]] | Vault, MCP, AI 연결, 정리 규칙 |
| (업무 흐름이 생기면 여기에 지도를 추가 — 예: 인프라와 서버, 콘텐츠와 보고서, 프로젝트 관계도) | |

## 사용법

- “어떤 프로젝트였지?”가 아니라 “무슨 업무 흐름이었지?”로 찾을 때 사용합니다.
- 세부 사실은 각 프로젝트의 `current-state`, `decisions`, `runbook`, `sessions` 문서에서 확인합니다.
- 과거 import 문서는 맥락용입니다. 현재 기준선은 `projects/*` 문서를 우선합니다.
